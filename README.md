# Fintech

Projeto de interface web para a aplicação Fintech, desenvolvido com foco em organização financeira pessoal. A proposta é oferecer uma experiência simples, responsiva e visualmente consistente para que o usuário possa acompanhar seus lançamentos financeiros, consultar gastos, visualizar proventos, definir metas e acessar dicas de educação financeira.

## Tecnologias utilizadas

- **HTML5**: estruturação das páginas e organização semântica do conteúdo.
- **CSS3**: definição de estilos globais, variáveis de cores e ajustes visuais complementares.
- **Tailwind CSS**: construção do layout, estilização dos componentes e responsividade da interface.
- **Git e GitHub**: versionamento do código-fonte e publicação do projeto em repositório remoto.
- **VS Code**: ambiente utilizado para desenvolvimento do projeto.

## Sobre o projeto

A aplicação representa uma interface de controle financeiro pessoal, permitindo ao usuário visualizar e organizar informações relacionadas a receitas, gastos, metas e lançamentos. O projeto prioriza uma identidade visual moderna, com tema escuro, uso de cores de destaque, componentes reutilizáveis e adaptação para diferentes tamanhos de tela.

O desenvolvimento foi realizado utilizando páginas HTML separadas dos arquivos CSS, mantendo a organização dos arquivos do projeto e garantindo que a aplicação possa ser executada diretamente pelo navegador através dos arquivos `.html`.

## Como executar o projeto

1. Clone o repositório:

```bash
git clone git@github.com:kleitonguedes/fiap-fintech.git
```

2. Acesse a pasta do projeto:

```bash
cd fiap-fintech
```

3. Abra o arquivo `.html` desejado diretamente no navegador.

Também é possível executar o projeto abrindo o arquivo `index.html` com duplo clique ou utilizando a extensão **Live Server** no VS Code.
